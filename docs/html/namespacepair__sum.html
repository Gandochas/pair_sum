pair_sum.pair_sum (   arr,
  k 
) 
Функция для проверки, существует ли в массиве `arr` пара элементов,
сумма которых равна заданному значению `k`.

Аргументы:
arr -- список целых чисел
k -- целое число, сумма которой нужно найти

Возвращает:
True, если существует хотя бы одна пара чисел в `arr`, сумма которых равна `k`.
False, если такой пары не существует.
